# Mocika Shield

Android APK 离线加固工具，桌面 GUI，三平台支持。

---

## 功能特性

- **DEX 加密保护**：对 APK 中的 DEX 文件进行压缩加密，防止静态反编译
- **防重打包**：证书指纹绑定加密链路，重打包后应用无法正常启动
- **多架构支持**：arm64-v8a / armeabi-v7a / x86 / x86_64
- **完全离线**：加固过程在本地完成，无需联网，不上传任何文件
- **内置签名工具**：GUI 内置 APK 签名标签页，支持自动清理签名产物

---

## 下载

前往 [Releases](../../releases) 页面下载对应平台的安装包。

| 平台 | 安装包 |
|------|--------|
| Linux | `.AppImage`（免安装）/ `.deb`（Debian/Ubuntu） |
| macOS | `.dmg` |
| Windows | `-setup.exe` |

---

## 使用说明

安装后直接打开，界面包含两个标签页：

- **加固**：拖入或选择 APK → 点击加固 → 等待完成 → 自动生成 `{name}_protected.apk`
- **签名**：拖入 APK + 密钥库文件 → 填写密码和别名 → 点击签名

> 加固后的 APK 未签名，需在签名标签页重新签名后方可安装。
